## 1. 设计概述

### 1.1 设计目标

本系统面向预制菜工厂的仓库管理人员，设计遵循以下原则：
- **高效性**：大屏宽屏布局(1920×1080)，信息密度适中，减少操作步骤
- **清晰性**：色彩编码区分状态，图标+文字双重识别
- **一致性**：Web端与小程序端共享统一的设计语言和配色体系
- **可访问性**：支持亮色/暗色双主题，适应不同光照环境

### 1.2 设计工具

- **设计参考**：Element Plus 设计规范
- **图标库**：Element Plus Icons (Web端)
- **图表库**：ECharts 5 (Web端) / Canvas 2D (小程序端)
- **字体**：系统字体栈（PingFang SC, Microsoft YaHei, -apple-system）

## 4. 间距系统

### 4.1 Web端间距

基于8px基础网格：

| Token | 值 | 用途 |
|-------|-----|------|
| `--space-xs` | 4px | 图标与文字间距 |
| `--space-sm` | 8px | 紧凑间距 |
| `--space-md` | 12px | 表单项间距 |
| `--space-lg` | 16px | 卡片内边距 |
| `--space-xl` | 20px | 卡片间距、页面内容区padding |
| `--space-2xl` | 24px | 大区块间距 |

页面内容区padding: `20px 24px`（Web端）；`30rpx`（小程序端）

### 4.2 小程序端间距

- 页面容器内边距: `0 30rpx`
- 卡片内边距: `30rpx`
- 卡片间距: `16rpx ~ 20rpx`
- 表单项间距: `20rpx`

## 7. 组件规范

### 7.1 按钮

**Web端：**
- 主按钮 `.btn-primary`：蓝绿色背景，白色文字，圆角6px
- 搜索按钮 `.btn-search`：蓝色调
- 危险按钮 `.btn-danger`：红色调
- 悬停时加深背景色，点击时`opacity: 0.85`

**小程序端：**
- 主按钮 `.btn-primary`：蓝色渐变(`#2563EB` → `#1D4ED8`)，白色文字，高度88rpx，圆角44rpx(胶囊形)
- 次要按钮 `.btn-secondary`：灰色背景
- 禁用态：灰色渐变(`#94A3B8` → `#CBD5E1`)

### 7.2 卡片

**Web端卡片 (`.card`)：**
- 白色背景，`border-radius: 12px`，边框`1px solid var(--border)`
- 默认阴影`--shadow-sm`，悬停时升级为`--shadow-md`
- 内边距：根据内容16-24px

**小程序端卡片 (`.card`)：**
- 白色背景，圆角16rpx，内边距30rpx
- 阴影`0 2rpx 12rpx rgba(0,0,0,0.04)`

### 7.3 状态标签 (`.tag`)

| 状态 | CSS类 | 背景色 | 文字色 |
|------|-------|--------|--------|
| 正常/已完成/启用 | `.tag-success` | `#F0F9EB` | `#67C23A` |
| 待检/待入库/待处理 | `.tag-warning` | `#FDF6EC` | `#E6A23C` |
| 报废/锁定/停用 | `.tag-danger` | `#FEF0F0` | `#F56C6C` |
| 已审核/待出库 | `.tag-info` | `#EFF6FF` | `#2563EB` |
| 草稿/默认 | `.tag-default` | `#F1F5F9` | `#64748B` |

样式：`font-size: 20rpx; padding: 6rpx 16rpx; border-radius: 8rpx`

### 7.4 表格

**Web端 (`.data-table`)：**
- 全宽布局，`border-collapse: collapse`
- 表头：`--background-deep`背景，11px大写字母，颜色`--foreground-muted`
- 数据行：悬停时背景变为`--primary-bg-hover`
- 单元格内边距：`10px 14px`

**小程序端 (`.order-card`)：**
- 卡片替代表格，每行一个卡片
- 卡片头：单号 + 状态标签
- 卡片体：标签-值对行（`.order-row`：`.order-label` + `.order-value`）

### 7.5 弹窗 (Modal)

**Web端：** 自定义模态框，背景遮罩`rgba(0,0,0,0.4)`，居中卡片
**小程序端：**
```css
.modal-overlay {
  position: fixed; top:0; left:0; right:0; bottom:0;
  background: rgba(0,0,0,0.4); z-index: 1000;
}
.modal-content {
  width: 580rpx; max-height: 80vh;
  background: #FFFFFF; border-radius: 24rpx;
}
.modal-wide { width: 640rpx; }
```

### 7.6 表单输入

**Web端：**
- 输入框：`9px 12px`内边距，`--input-border`边框，聚焦时蓝色边框+外发光
- 文本域：最小高度80px

**小程序端：**
- 输入框：高度64rpx，`#F8FAFC`背景，圆角12rpx，内边距`0 20rpx`
- 文本域：高度120rpx，同上背景和圆角

### 7.7 图表 (ECharts / Canvas)

**Web端 (ECharts 5):**
- 折线图：平滑曲线 + 半透明填充区域
- 柱状图：渐变色柱体
- 双Y轴图：温度(℃)+湿度(%)，零线对齐
- 进度条：绿色(<25%) → 橙色(<50%) → 琥珀色(<75%) → 红色(≥75%)

**小程序端 (Canvas 2D):**
- 折线图：带圆点标记 + 图例
- 柱状图：绿色渐变柱体，横向滚动支持
- 双轴柱状图：红色(温度)+蓝色(湿度)
- 进度条：水平柱状条，颜色梯度同Web端

## 9. 主题切换 (Web端)

### 9.1 实现机制

使用 CSS 自定义属性 + `data-theme` 属性方案：

```css
:root { /* 亮色主题变量 */ }
[data-theme="dark"] { /* 暗色主题覆盖 */ }
```

### 9.2 切换方式

- 顶栏右侧太阳/月亮图标按钮
- 点击即切，无过渡动画(即时生效)
- 状态持久化到 `localStorage('app-theme')`
- 技术实现：`src/composables/useTheme.js` (Vue 3 Composition API)

### 9.3 暗色主题特点

- 深色背景(`hsl(160,8%,8%)`)，浅色文字(`hsla(0,0%,93%)`)
- 卡片背景`hsl(160,5%,15%)`，更深阴影
- 侧边栏渐变更低调
- 状态色彩亮度提高，确保可读性
- 图表配色自动适配

## 10. 响应式与适配

### 10.1 Web端

- **目标分辨率**：1920×1080 (16:9宽屏)
- **最小支持**：1366×768
- 侧边栏折叠功能适配小屏

## 11. 交互规范

### 11.1 加载状态

- **Web端**：骨架屏(Shimmer动画)、路由加载动画(跳动方块)、按钮loading禁用
- **小程序端**：`wx.showLoading({ title: '加载中...' })` + `wx.hideLoading()`；空状态提示

### 11.2 反馈提示

- **操作成功**：绿色Toast (1500ms)
- **操作失败**：红色Toast + 错误信息
- **网络错误**：统一"网络错误"提示
- **确认操作**：`wx.showActionSheet` / `wx.showModal` / Web端自定义确认对话框

### 11.3 导航

- **Web端**：Hash路由 (`/#/home`)，`router.push`/`router.replace`
- **小程序端**：`wx.navigateTo`(保留历史)、`wx.redirectTo`(替换)、`wx.reLaunch`(重启)、`wx.switchTab`(Tab切换)

### 11.4 权限控制

- **Web端路由守卫**：`router.beforeEach`检查token；`meta.requirePosition: '主管'`限制管理页面
- **小程序端**：按钮/功能根据`userInfo.position`条件渲染(`wx:if="{{isAdmin}}"`)
- 三种角色：主管(全部权限)、仓库员工(出入库执行)、质检员(质检操作)
