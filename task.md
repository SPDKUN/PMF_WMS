  关键技术特性

  - API对接: 通过 utils/api.js 封装了全部后端24+个接口，遵循 { code: 200, data: ..., msg: ... } 响应格式
  - JWT认证: 自动在请求头添加 Authorization: Bearer <token>，token过期自动跳转登录页
  - 服务器地址: 当前配置为 http://localhost:8088，可在 app.js 的 globalData.baseUrl 中修改
  - 5个底部导航: 首页、管理、查询、任务、我的

  使用前注意事项

  1. Base URL: 在 app.js 中修改 baseUrl 为实际后端服务器地址（需支持HTTPS且已在小程序后台配置合法域名）
  2. Tab图标: 当前TabBar为纯文字模式。如需图标，在 app.json 的 tabBar.list 中为每项添加 iconPath 和
    selectedIconPath（需81x81px PNG文件）
  3. 后端服务: 确保 Spring Boot 后端（端口8088）已启动且数据库 pmf（密码：123456）可连接
