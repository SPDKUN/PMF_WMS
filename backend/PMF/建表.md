| `status` | TINYINT | ✓ | 1 | 状态：0=禁用, 1=启用 |

**唯一键:** `username`

## ER 关系概要

```
user ─────────────────────────────────────────────────────────────────────────────
  │
  ├── warehouse ──── zone ──── location
  │       │            │           │
  │       └────────────┴───────────┘ (FK 引用)
  │
  ├── goods ──── batch ──── inventory
  │     │          │
  │     └──────────┴── work_task
  │
  ├── inbound_order_head ──── inbound_order_detail
  ├── outbound_order_head ─── outbound_order_detail
  ├── adjustment_order_head ─ adjustment_order_detail
  ├── inventory_check_head ── inventory_check_detail
  ├── quality_check_head ──── quality_check_detail
  ├── defective_handling_head ─ defective_handling_detail
  │
  ├── temperature_humidity_record
  └── operation_log
```

**核心流程:** 货物(goods) → 批次(batch) → 库存(inventory) → 库位(location) → 库区(zone) → 库房(warehouse)

**单据流程:** 入库 → 质检 → 库存管理（调整/盘点） → 出库 → 不合格处理
