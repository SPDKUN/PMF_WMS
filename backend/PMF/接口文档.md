# WMS 仓储管理系统接口文档 v2.0

> 基于新数据库表结构全面重构（2026-07-10）
> 项目根路径：`http://localhost:8088`

所有接口返回统一格式：
```json
{ "msg": "操作成功", "code": 200, "data": null }
```

---

## 1. 认证管理 `/auth`

### 1.1 用户登录
- **路径：** `GET /auth/login`
- **参数：** `username` (query), `password` (query)
- **返回示例：**
```json
{
  "msg": "操作成功", "code": 200,
  "data": { "user_id": 1, "username": "admin", "real_name": "管理员", "phone": "13800000000", "department": "仓储部", "position": "经理", "status": 1 },
  "token": "eyJhbGciOi..."
}
```

## 2. 用户管理 `/user`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/personnel` | 查询用户列表 |

## 变更说明（相比 v1.0 旧版）

| 旧接口 | 变更 |
|--------|------|
| `/book` | 删除 |
| `/inventoryRecord` | → `/inventory`（字段重构） |
| `/inboundOrder` | → `/inboundOrderHead` + `/inboundOrderDetail`（头+明细拆分） |
| `/outboundOrder` | → `/outboundOrderHead` + `/outboundOrderDetail`（头+明细拆分） |
| `/inspectionReport` | → `/qualityCheckHead` + `/qualityCheckDetail`（头+明细拆分） |
| `/temperatureRecord` | → `/temperatureHumidityRecord`（增加湿度字段） |
| `/checkTask` | → `/workTask`（字段重构为通用任务表） |
| (新增) | `/adjustmentOrderHead` + `/adjustmentOrderDetail` |
| (新增) | `/inventoryCheckHead` + `/inventoryCheckDetail` |
| (新增) | `/defectiveHandlingHead` + `/defectiveHandlingDetail` |
| (新增) | `/operationLog` |
